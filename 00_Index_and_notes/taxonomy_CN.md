# 分类法：LLM 人格化与意识邻近建模

| 类别 | 核心问题 | 主要证据/方法 | 对综述的用途 |
|---|---|---|---|
| 01 综述与框架 | 如何定义 persona、personalization、role-playing agent、意识邻近能力？ | Survey、taxonomy、risk framing | 引言和整体结构 |
| 02 Persona / Role-playing | LLM 如何保持稳定角色、身份、说话风格和价值立场？ | Character agent、role fidelity、multimodal role-playing、persona evaluation | 人格化的“显性身份层” |
| 03 Personalization | LLM 如何适配具体用户，而不是抽象人群？ | 用户画像、偏好学习、交互式对齐、decoding-time alignment | 个体化与用户模型 |
| 04 Memory / Self-continuity | 模型如何跨会话保持“我是谁、你是谁、我们经历过什么”？ | 长期记忆 benchmark、profile update、agent memory system | 自我/关系连续性 |
| 05 Human-like / Social simulation | LLM 是否能模拟真实人的心理特征、群体行为、社会约定？ | Self-report grounding、Turing test、psychometrics、multi-agent populations | human-like 行为证据和社会模拟 |
| 06 Self-model / Introspection | 模型能否描述自身行为、内部状态、不确定性来源或主观体验报告？ | Behavioral self-awareness、activation introspection、self-referential processing、uncertainty attribution | 意识邻近机制主线 |
| 07 Evaluation / Validity / Safety | 人格化和意识邻近评测是否可靠？会不会放大偏见或制造伪证据？ | ToM critique、persona stereotype tests、validity critique、motivated reasoning | 反证、限制与安全章节 |
| 08 Theory background | 哪些意识科学理论可以转化为 AI 指标？ | GWT、HOT、predictive processing、attention schema 等理论指标 | 只作理论边界，不作方法主线 |

建议最终综述标题/研究问题不要写成“如何把意识放进 LLM”，而写成：

> How to model consciousness-adjacent functional components in LLM agents: persona, memory, self-model, personalization, introspection, and social cognition.
